# How To Import LAN Party

## Dev Plugin Path

`Z:\LanParty\LanParty\bin\x64\Debug\LanParty.dll`

## Add The Plugin In XIVLauncher

1. Launch FFXIV through XIVLauncher.
2. Open Dalamud settings with `/xlsettings`.
3. Go to `Experimental`.
4. Add `Z:\LanParty\LanParty\bin\x64\Debug\LanParty.dll` to `Dev Plugin Locations`.
5. Open the plugin installer with `/xlplugins`.
6. Go to `Dev Tools > Installed Dev Plugins`.
7. Enable `LAN Party`.
8. Run `/lanparty` in game to open the main window.
9. Use `/lanparty config` if you want the settings window directly.
